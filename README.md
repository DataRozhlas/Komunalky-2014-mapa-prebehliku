# Mapa přeběhlíků komunálních voleb 2014

Celý článek na webu Rozhlasu: [Kdo ve vaší obci kandiduje za jinou stranu než v minulých volbách?](http://www.rozhlas.cz/zpravy/data/_zprava/data-kdo-ve-vasi-obci-kandiduje-za-jinou-stranu-nez-v-minulych-volbach--1405616)

Mapa zvýrazňující obce, ve kterých kandidáti změnili stranu, za kterou kandidují.

## Instalace

    npm install -g LiveScript@1.2.0
    npm install
    slake deploy

Hlavní stránka je v www/index.html

Mapové dlaždice jsou generovány z SVG v adresáři data pomocí [SVG Mapperu](https://github.com/economia/svg-mapper).
